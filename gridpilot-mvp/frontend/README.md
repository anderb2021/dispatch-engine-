# GridPilot Frontend

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Pages included

- Landing page
- Tesla connection CTA placeholder
- Passive rewards explanation
- Demo dashboard
- Flexibility stats
- Recent dispatch events
